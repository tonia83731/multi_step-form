# Multi Step Form

This is a multi-step from practice using react, vite, typescript.

## Environment

- node @20.11.1
- react @18.2.0
- tailwindCSS @3.4.1
- typescript @5.2.2

## Getting Start

1. Clone the project to local, enter:

```
git clone https://github.com/tonia83731/multi_step-form.git
```

2. Go into the project(cd), later enter to install npm:

```
npm install
```

3. Start the project by enter:

```
npm run dev
```

4. If you wish to end the project, enter:

```
ctrl + c
```
